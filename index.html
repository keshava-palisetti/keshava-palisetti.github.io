Brief Description :
The user has to register first to make a mail approval request. After registration, the user gets a userid and the password while creating the account, which are useful to login and make a approval request up to maximum of 3 requests.
After logging in, user can track his/her approval request in the profile page and will be approved in 24 hours if the details are valid.
